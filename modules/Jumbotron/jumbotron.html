Terrific Micro Grunt

Based on Terrific Micro but using grunt for concatenation and minification.

Features:
  • Module Generator (grunt-terrific-modules)
    • grunt terrific_modules
  • Integrated QUnit tests
    • grunt build-tests
    • url: /cache/tests.html
  • LESS support
    • Build: grunt build
    • With watch task: grunt
    • concatenation and compiling: grunt build-styles-fast
    • concatenation, compiling and minification: grunt build-styles
  • CSS and JS concatenation
    • Build: grunt build
    • With watch task: grunt
    • concatenation and compiling less: grunt build-styles-fast
    • concatenation js: grunt build-scripts-fast
  • CSS and JS minification
    • Build: grunt build
    • With watch task: grunt
    • concatenation, compiling and minification less: grunt build-styles
    • concatenation and minification js: grunt build-scripts
  • dot compiler
    • Build: grunt build
    • With watch task: grunt
    • concatenation and compiling: grunt build-dot-fast
    • concatenation, compiling and minification: grunt build-dot
  • handlebars compiler
    • Build: grunt build
    • With watch task: grunt
    • concatenation and compiling: grunt build-hbs-fast
    • concatenation, compiling and minification: grunt build-hbs
  • Sprite generator with retina support
    • grunt sprites
  • JsHint
    • grunt jshint
  • String search
    • grunt search
  • Complexity calculation
    • grunt complexity
  • YUIDoc JavaScript documentation
    • grunt yuidoc
    • url: /cache/jsdoc/
  • Export pages
    • Build and export: grunt export
    • url: /cache/exported/
  • W3C markup validator
    • export and validate: grunt w3c
    • just validate exported pages: grunt validate
